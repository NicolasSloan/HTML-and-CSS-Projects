What is an article tag?

Articles are used to group content together that is related, and it should also contain a heading.
It should also be self-contained, meaning it can be separated from the rest of the document and still be meaningful

Details and summary tag

The details tag is used to specify additional data that the user can open and close.
It is closed by default. The summary tag is used to define a visible heading for the details element

The time tag is used to define a specific time, such as 14:00.
You can then use the datetime attribute to translate the time into a machine readable format.
This allows browsers to offer date reminders through calenders, or help search engines produce smarter results
